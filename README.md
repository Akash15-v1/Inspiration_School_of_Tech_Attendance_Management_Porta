# Serverless Student Attendance Management System 
The Serverless Student Attendance Management System is a cloud-based application developed using Amazon Web Services (AWS) to automate attendance management in educational institutions. Traditional attendance systems require significant manual effort and infrastructure maintenance, whereas this project leverages serverless computing to eliminate server management while ensuring scalability, availability, and cost efficiency.
The application uses Amazon API Gateway to expose REST APIs, AWS Lambda for backend processing, Amazon DynamoDB for storing attendance records, Amazon Cognito for user authentication, and Amazon S3 for hosting the web application.
